## 🎯 The Problem

Content creators face a daily struggle:
- ✍️ Writing for LinkedIn? Professional tone
- 🐦 Posting on Twitter? Short and punchy
- 📸 Instagram caption? Engaging with hashtags
- 📝 Blog post? Long-form and detailed
- 📧 Email newsletter? Conversational

**Same idea. Five different formats. Hours of work.**

## ✨ The Solution

Content Forge uses AI to transform **one topic** into **five platform-optimized pieces** in seconds.

```
Input: "AI in healthcare"
Output: 
  💼 LinkedIn post (professional, 300 words)
  🐦 Twitter thread (punchy, 280 chars)
  📸 Instagram caption (engaging + hashtags)
  📝 Blog article (comprehensive, 2000 words)
  📧 Email newsletter (conversational, 500 words)
```

## 📝 API Documentation

### POST `/api/generate`
Generate content for all platforms

**Request:**
```json
{
  "topic": "AI in healthcare",
  "tone": "professional"
}
```

**Response:**
```json
{
  "success": true,
  "topic": "AI in healthcare",
  "tone": "professional",
  "results": {
    "linkedin": { "content": "...", "metadata": {...} },
    "twitter": { "content": "...", "metadata": {...} },
    "instagram": { "content": "...", "metadata": {...} },
    "blog": { "content": "...", "metadata": {...} },
    "email": { "content": "...", "metadata": {...} }
  }
}
```

### POST `/api/generate/:platform`
Generate for specific platform

### GET `/api/platforms`
Get supported platforms and metadata
